# 📌 Nome do Projeto
PROJETO BOM
## 📚 Descrição do Projeto
Este projeto foi criado para:
praticar os conceitos fundamentais de desenvolvimento de servidores e manipulação de dados.

O objetivo do sistema é:permitir que usuários criem, visualizem, editem e excluam tarefas (CRUD) através de rotas HTTP.

## 🚧 Status do Projeto
(x) Em desenvolvimento
( ) Finalizado
## 🔨 Funcionalidades
- Cadastro de Usuários: Criação de conta para acessar o sistema.
- Gerenciamento de Tarefas: Endpoint para salvar e listar atividades.
- Autenticação: Proteção de rotas para usuários logados.
## 💻 Tecnologias Utilizadas
- Node.js: Ambiente de execução para o JavaScript no servidor.
- Express: Framework para facilitar a criação de rotas e middlewares.
- SQLite/MongoDB: Banco de dados para persistência das informações
